üìã About This Project

Purpose: Develop an integrated mentoring system that addresses three critical challenges in the CAMD PhD program:

  • Irregular faculty-student meetings
  • Poor communication and unclear expectations
  • Milestone confusion leading to poor performance on exams and defenses

Approach: Six integrated forms using WordPress/Formidable Forms that support students, faculty, and administrators throughout the 5-year doctoral journey.

Status: Design complete, specifications ready, mockup drafted. Implementation pending approval.

The Six Components

  1. Individual Development Plan (IDP) - Longitudinal goal tracking
  2. Lab Manual - Faculty expectations (created once)
  3. Advising Compact - Student-advisor agreement
  4. Summer Check-in - Developmental reflection (August, CV required)
  5. Fall Check-in - Progress review (December)
  6. Annual Review (360¬∞) - Comprehensive mutual assessment (April)

üéØ Key Features

‚úì Data Integration

IDP data flows into Compact. Check-ins reference goals. Annual Reviews build on previous feedback. No re-entering same information.

‚úì Discipline-Aware

Forms adapt for STEM, Humanities, and Practice-Based research. Students see relevant prompts based on their methodology.

‚úì 360¬∞ Feedback

‚úì Privacy Built-In
